# OJDropdown

Custom, accessible, and keyboard-friendly dropdown component built with vanilla JavaScript.

## Features

- Fully **keyboard navigable**: ArrowUp, ArrowDown, Enter, Escape, Tab.
- Supports **single selection** with visual highlighting.
- Smooth open/close animations.
- Customizable styling via **CSS variables**.
- Automatically updates button label when an option is selected.
- Self-contained **inline tests** with colored console output for pass/fail.

## Installation
Simply put the dropdown_coding_challenge.html on your computer and run using firefox or chrome

##  Testing

Inline self-contained tests are included in the script:

- Logs green text for passing tests.
- Logs red text for failing tests.

### Covers:

- Initial state
- Option selection
- Open/close behavior
- Keyboard navigation
- DOM updates
- Focus handling
